# gyakukyujin quick index

面談現場快速定位用（`id` + 一句用途）。

- `1`：先讓對方定義你現在的 fit。
- `2`：逼對方說出 why you / 半年期待值。
- `3`：確認你的職能落點（product/structure/implementation）。
- `4`：問現在團隊真正缺什麼人。
- `5`：讓對方想像你 join 後的進場方式。
- `6`：先問對方對你的顧慮與待確認點。
- `7`：問跨域型人才在他們那邊怎麼被接住。
- `8`：輕微相親感，請對方說吸引你的理由。
- `9`：問新卒可塑性與成長路徑。
- `10`：問自走的責任邊界與文化。
- `11`：對方說你偏 product/architecture 時的追問。
- `12`：對方說先單一職能時的追問。
- `13`：對方說要看 coding 時的追問。
- `14`：對方說缺業務技術橋接人時的追問。
- `15`：對方覺得你特別但不知怎麼用時的追問。
- `16`：對方問你重視什麼時的標準答法。
- `17`：收尾，留下被追者餘味。
- `18`：柔和確認他們會怎麼接你這型。
- `19`：微壓式提問，請對方更細緻說明。
- `20`：對方回答抽象時，拉回具體場景。
- `21`：問是否願意為你這型調整 role。
- `22`：問這類候選人在內部如何被討論。
- `23`：把自己放進半年後理想圖。
- `24`：問 mentor / 接應者機制（新卒關鍵）。
- `25`：對方說入社後再調整時的追問。
- `26`：對方說你跨太多時的收斂追問。
- `27`：對方說重視自走時，定義自走含義。
- `28`：對方說重視 user/business 時，問痛點切入。
- `29`：微曖昧確認：對方是否已在認真想像你的位置。
- `30`：追擊，讓對方自己說出「為何是你」。
- `31`：高級挑選感：你也在看長期成長位點。
- `32`：氣氛好時丟的高級判斷句。
- `33`：收尾追加，給對方後續思考作業。
- `34`：超短補句：停頓後柔和接續。
- `35`：超短補句：請對方繼續展開。
- `36`：超短補句：柔軟表達你也在挑 fit。
- `37`：超短補句：輕微被追者感。
- `38`：超短補句：引導對方用公司語言口說你。
